### Local / Server Deployment

#### Prerequisites:

- Node 20.x

To get started you need to clone the repo

```bash
git clone https://github.com/Parcoil/lunaar.org
cd lunaar.org
```

Next you need to enable pnpm

```bash
corepack enable pnpm
```

Now install the required packages

```bash
pnpm install
```

Now start lunaar

```bash
pnpm start
```

#### Updating Lunaar

```bash
cd lunaar.org
git pull --force
```
